Hello World

May 14th, 2013

O blog dos DPUM está oficialmente no ar.

Para já, este blog será o único meio de comunicação para o clube, sendo adicionadas outras plataformas em função da adesão às actividades. Vejam isto como achievements:

Achievement Prize
2 semanas blog (pfac@github), GitHub repo (pfac@github:dpum)
4 semanas blog (cesium@github), GitHub repo (cesium@github:dpum)
6 semanas Email account, GitHub account, Twitter account
… Coming (not so) soon

Isto serve para mostrar que o investimento neste clube por parte da organização será proporcional ao investimento por parte dos participantes. Se deixarem de participar, basicamente isto deixa de existir.

Sessões

Nos próximos dias deverão aparecer aqui novos posts relativos ao rescaldo das primeiras sessões. Usem a feed RSS para receberem as actualizações deste blog.

Para os interessados que ainda conservem alguma função motora e intelectual durante estas Monumentais Festas do Enterro da Gata, familiarizem-se com o problema B ( "The Toxic Clouds of Proxima III" ), e claro, tentem resolver. Este problema é bastante versátil, e vai ser usado novamente nas próximas sessões para explicar outros algoritmos.

Próxima Sessão: Quinta, 16 de Maio, 14h30 (hora a confirmar)

Continuação de bons festejos, e tentem não ficar reduzidos a code monkeys.
